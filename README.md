# ConsumiendoApi-ionic

Realiza una aplicacion tipo SIDEMENU que consuma al menos 3 servicios diferentes, use el menu para poder intecambiar de opciones y probar los servicios. 
* RamdonUsers
* DGII RNC
* pokeapi
